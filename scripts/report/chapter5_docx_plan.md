# 第五章 docx 生成计划

## 目标

用 poi-tl 从模板生成第五章 docx，包含全部正文、表格、图表。

## 流程

```
Step 1: 创建 poi-tl 模板
├── 从提纲提取11节结构
├── 每节插入 {{变量}} 占位符
├── 表格用 {{#table}} 循环
├── 图片用 {{@image}} 插入
└── 输出: templates/chapter5_template.docx

Step 2: 准备 JSON 数据
├── 正文文字（每节1-3段）
├── 表格数据（表5-1~5-12）
├── 图片路径（7张图）
└── 输出: data/chapter5_data.json

Step 3: 渲染生成
├── mvn exec:java 调用 poi-tl
└── 输出: output/第五章_实验验证与结果分析.docx

Step 4: 检查与修正
├── 检查格式（标题、字体、行距）
├── 检查表格（数据正确性）
├── 检查图片（是否正确插入）
└── 必要时调整模板重新生成
```

## 写作口径检查清单

| 节 | 口径要求 | 检查点 |
|---|---------|--------|
| 5.1 | 不写"水面人体目标数据集" | 标题和正文用"视觉定位基础链路" |
| 5.5 | 不写"复现优于/低于论文" | 用"链路测试结果" |
| 5.6 | 不简单归因算法失效 | 说明参考图质量差异 |
| 5.7 | 标注小样本量 | 50-70°标注n=22 |
| 5.8 | 聚焦三个核心对比 | Top-N/RoMa/噪声 |
| 5.9 | 写"分析框架" | 不写已完成实测 |
| 5.10 | 按原因分类 | 视角退化/参考图差异/检测框联动 |

## 数据来源映射

| 内容 | 来源文件 |
|------|---------|
| 正文文字 | 根据提纲撰写 |
| 表5-1~5-12 | `素材/实验数据/第五章表格数据.md` |
| 图5-1~5-8 | `素材/截图/fig5_*.png` |
| 先验噪声数据 | 论文 notes.md（标注来源） |
| 分场景数据 | `素材/实验数据/results_HIGH.csv` |
